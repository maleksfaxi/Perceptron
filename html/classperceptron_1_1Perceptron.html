Detailed Description

    Perceptron algorithm class.
    The perceptron is a binary classifier of the form  y^hat=h(x)=sign(w^tx+b) with b the bias and w a vector of real-valued weights.
    __init__() initializes the  Initializes the perceptron algorithm with specified learning rate and maximum iterations.
    activation() applies the Heaviside activation function to the input.
    train() trains the perceptron by updating weights and bias to learn a decision boundary that separates classes.
    predict() predicts the class labels for a given input.
